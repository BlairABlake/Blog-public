Standard I/O Streams

When the main function is invoked, it already has three streams open and available for use. These are standard input and output channel that have been established for the process.

  • stdin: the standard input stream
  • stdout: the standard output stream
  • stderr: the standard error stream

These streams are, by default, connected to the shell from which the code is being running.

In the GNU C library, these standard streams are normal variables, which means you can specify the streams by modifiying the variables.

The standard input from a file

stdin = fopen("file name". "r");

The standard output to a file

stdout = fopen("file name", "w");

Type Conversion and Type Casting

The type conversion from the higher to lower level is prohibited.

The hierachy

lower_type y = (lower_type)x;

Functions

function declaration

return_type function_name(data_type v1, data_type, v2, ...);

Placing the function declaration prior to its use enables the compilers to make a check, so it’s basically place in a header file.

function definition

return_type function_name(data_type v1, data_type v2, ...) 
{
    ...
    return ;
}

There are two ways the arguments are passed to a function.

call by value

The called function creates new variables to store the value of the arguments. This is a default behavior.

call by reference

The called function just receives a reference to the data. To indicate an arguemtn is passed using call by reference. an asterisk * is placed after the type in the arguments.

return_type function_name(data_type *v1, data_type, v2, ...);

In this code, v1 will be passed in call by value, and v2 will be passed in call by reference.

Pointers

Every variable in C has a value and also a memory location(adress). The value and the address are called rvalue and lvalue respectively.

declaring pointer variables

data_type *ptr_name;

For example

int *ptr;

In this case, ptr is a lvalue and *ptr is a rvalue. When assigning a value to a variable, lvalue must be assigned to a pointer variable, and rvalue must be assigned to a normal variable.

ptr = &x;

*ptr = x;

Strings

In C, a string is a null terminated character array. null terminated means, the last elemtn is a null character “\o”.

char str[] = "Hello";

Then str has a six elements h, e, l, l, o, and ‘\o’

reading strings from stdin stream

scanf("%[width]s", str);

width specify the maximum number of characters to read.

get a byte form stdin stream

getchar()

read bytes from stdin stream until a newline is read

gets()

writing strings to stdout stream

printf("%s", str);

put a byte on a stdout stream

putchar(str[i]);

put a string on stdout stream

Unions

A union is a collection of variables of different types, but only store one field at one time. The size of union is the largest size of fields. With unions, you can store the different type of values to the same memory address.

union union_name 
{
    data_type name;
    ...
};

All fields point to the same data.

union Point
{
    int x;
    int y;
    long n;
}

sizeof(Point) == sizeof(long)
Point P;
p.x = 10;
p.n = 20;
p.x == 20 //true
p.y == 20 //true
